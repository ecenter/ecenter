E-Center classes

WYSIWYG filter asks us to restrict the classes available to regular users. The theme provides several ecenter- prefixed helper classes so users can compose pages using the normal editing mechanisms.

.ecenter-introduction

E-Center introduction text: To be used to preface pages.

.ecenter-highlight

A class for highlighting things.

.ecenter-pad

Pad an element, like a highlighted paragraph.
